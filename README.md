# Electrical Shop Inventory Management System

This project is an inventory management system for an electrical shop. It consists of a React frontend and a Node.js/Express backend.

## Project Structure

- `frontend/`: The React client application.
- `backend/`: The Node.js server API.

## Getting Started

### Prerequisites

- Node.js installed

### Setup

1. **Backend**
   ```bash
   cd backend
   npm install
   npm start
   ```

2. **Frontend**
   ```bash
   cd frontend
   npm install
   npm start
   ```
